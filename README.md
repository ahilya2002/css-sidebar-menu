# CSS Sidebar Menu

A responsive sidebar navigation menu built with pure HTML and CSS — no JavaScript required.

## ✨ Features

- Toggle sidebar open/close using a pure CSS checkbox hack
- Glassmorphism effect (frosted glass background)
- Font Awesome icons for menu items and social links
- Google Fonts (Poppins) integration
- Social media links section

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome 6
- Google Fonts (Poppins)
